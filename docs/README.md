# Enquadramento Documentation

This folder documents the project structure, data flow, and how the site is assembled from database content and curated categories.

## Docs map

- `docs/architecture.md` - entrypoint, core components, and request flow diagram
- `docs/feed-flow.md` - news ingestion flow, sources definition, and fetch pipeline
- `docs/data-model.md` - database tables and relationships
- `docs/categories-and-home.md` - category config, training mapping, and homepage sections
- `docs/routing-and-views.md` - routes, controllers, and views
- `docs/share-links.md` - share and redirect tracking flow
- `docs/operations.md` - migrations, fetch, and operational notes

## Diagram notes

Diagrams are written in Mermaid. Most Markdown viewers render them automatically; if yours does not, use a Mermaid-enabled preview.
